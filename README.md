## Usage

```console
$ drunken-bishop hello world
+-----------------+
|                 |
|       o         |
|      . X . E    |
|       O + =     |
|      o S + *    |
|     .   . =     |
|                 |
|                 |
|                 |
+-----------------+
```

## Install

### Using go

```console
$ go get -u moul.io/drunken-bishop
```

### Using brew

```console
$ brew install moul/moul/drunken-bishop
```
